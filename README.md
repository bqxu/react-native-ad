### 更新模块 🍗

怎么查看当前版本？执行 `yarn list | grep "react-native-ad"`，将会输出你当前项目安装的 react-native-ad 版本

然后可以使用 npm update 来更新模块

```
$ npm update react-native-ad
```

或者使用 yarn upgrade 来更新

```
$ yarn upgrade react-native-ad
```
